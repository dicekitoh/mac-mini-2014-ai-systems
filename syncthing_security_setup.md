# Syncthing セキュリティ設定手順

## GUI認証設定方法

1. **設定画面を開く**
   - 右上の「アクション」→「設定」をクリック

2. **GUI タブを選択**

3. **認証情報を設定**
   - **GUIユーザー名**: `admin`（または任意のユーザー名）
   - **GUIパスワード**: 強力なパスワードを設定
   
   推奨パスワード例: `Sync@2025#Secure`

4. **保存して再起動**
   - 「保存」をクリック
   - Syncthingが自動的に再起動します
   - 再度アクセスする際は設定したユーザー名とパスワードが必要になります

## 追加のセキュリティ設定（推奨）

### リッスンアドレスを制限
現在の設定では外部からアクセス可能になっているため、ローカルネットワークのみに制限することをお勧めします：

1. 「設定」→「GUI」タブ
2. 「GUIリッスンアドレス」を以下のいずれかに変更：
   - `127.0.0.1:8384` (ローカルマシンのみ)
   - `192.168.3.43:8384` (特定のIPのみ)

## 設定後の確認
- ブラウザを再読み込みしてログイン画面が表示されることを確認
- 設定したユーザー名とパスワードでログインできることを確認

この設定により、あなたのファイルが不正アクセスから保護されます。
