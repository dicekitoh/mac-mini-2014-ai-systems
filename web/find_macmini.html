MacMini2014 IPアドレス検索方法

MacMini2014の情報:
MACアドレス: 98:10:e8:f1:98:b4
ホスト名: macmini2014
通常のIPアドレス: 192.168.3.43

方法1: 直接アクセス（IPアドレスが分かっている場合）

以下のリンクから直接アクセスできます：

http://192.168.3.43/ IPステータスページ

方法2: ホスト名でアクセス

ローカルネットワーク内では、ホスト名でアクセスできる場合があります：

http://macmini2014/ http://macmini2014.local/

方法3: コマンドラインで検索

Windows:

ping macmini2014
arp -a | findstr "98-10-e8-f1-98-b4"

Mac/Linux:

ping macmini2014.local
arp -a | grep "98:10:e8:f1:98:b4"

方法4: Pythonスクリプトで検索

find_macmini2014.py をダウンロードして実行：

python find_macmini2014.py

方法5: ルーター管理画面から確認

ルーターの管理画面にアクセスして、接続されているデバイス一覧を確認：

  1. ルーター管理画面にアクセス
  2. ユーザー名: user / パスワード: ***REMOVED*** でログイン
  3. 「接続デバイス一覧」や「DHCPクライアント」を探す
  4. MACアドレス 98:10:e8:f1:98:b4 を探す
ヒント:
• IPアドレスが変わった場合は、まず方法2（ホスト名）を試してください
• それでもアクセスできない場合は、方法3〜5で新しいIPアドレスを探してください
• 見つかったIPアドレスは、ブックマークに保存しておくと便利です
