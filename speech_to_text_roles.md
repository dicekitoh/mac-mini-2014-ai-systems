# Google Cloud Speech-to-Text API 必要なロール

## 推奨ロール（どれか1つ）

### 1. **Speech-to-Text 編集者** (推奨)
- **ロール名**: `roles/speech.editor`
- **説明**: Speech-to-Text APIの全機能を使用可能
- **検索方法**: "speech" で検索

### 2. **Speech-to-Text ユーザー**
- **ロール名**: `roles/speech.client`
- **説明**: 音声認識の実行に必要な最小限の権限
- **検索方法**: "speech" で検索

### 3. **基本ロール** (簡単だが権限が広い)
- **編集者**: `roles/editor`
- **説明**: プロジェクト全体の編集権限（権限が広すぎる場合がある）

## 設定手順

1. サービスアカウントのページで「権限」または「ロールを付与」をクリック
2. 検索ボックスに「**speech**」と入力
3. 以下のいずれかを選択：
   - **Speech-to-Text 編集者** (推奨)
   - **Speech-to-Text ユーザー**

## 検索のコツ
- 「speech」で検索すると Speech 関連のロールのみ表示
- 「音声」では出てこないので英語で検索

## 最小限の権限
音声文字起こしのみ行う場合は「**Speech-to-Text ユーザー**」で十分です。
