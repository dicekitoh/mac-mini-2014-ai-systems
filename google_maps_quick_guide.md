# Google Maps API クイック設定ガイド

## 🚀 5分で完了！最短手順

### 1. Google Cloud Console を開く
```
https://console.cloud.google.com/
```

### 2. プロジェクト作成
- 左上の「プロジェクトを選択」→「新しいプロジェクト」
- プロジェクト名: `route-optimizer`（任意）
- 「作成」クリック

### 3. APIを有効化（重要！）
左メニュー「APIとサービス」→「ライブラリ」で以下を検索・有効化：

1. **Geocoding API** - 住所→座標変換
2. **Distance Matrix API** - 距離・時間計算
3. **Places API** - 場所検索（オプション）
4. **Maps JavaScript API** - 地図表示（オプション）

各APIページで「有効にする」ボタンをクリック

### 4. APIキー作成
1. 「APIとサービス」→「認証情報」
2. 「+ 認証情報を作成」→「APIキー」
3. 作成されたキーをコピー（例: `AIzaSy...`）

### 5. 請求アカウント設定（必須）
1. 左メニュー「お支払い」
2. 「請求先アカウントをリンク」
3. クレジットカード情報入力
   - 月$200まで無料（約28,500リクエスト）
   - 自動課金されません（上限設定可能）

### 6. APIキー設定（コマンド実行）
```bash
# 設定スクリプト実行
python3 /home/fujinosuke/projects/setup_google_maps_api.py

# APIキーを貼り付けて、設定方法を選択
```

### 7. 動作確認
```bash
# 環境変数を有効化
source ~/.bashrc

# テスト実行
python3 /home/fujinosuke/projects/route_optimizer_tsp.py
```

## ⚠️ よくあるエラーと対処法

### REQUEST_DENIED エラー
- **原因1**: 請求アカウント未設定 → お支払い設定を完了
- **原因2**: API未有効化 → 上記4つのAPIを有効化
- **原因3**: APIキー制限 → 一旦制限を解除してテスト

### OVER_QUERY_LIMIT エラー
- 1日の無料枠超過 → 翌日再試行

## 💡 ヒント

1. **新規アカウント特典**: 最初の90日間は$300のクレジット付き
2. **予算アラート設定**: 念のため月$10等で設定推奨
3. **APIキー制限**: 本番運用時はIPアドレス制限推奨
